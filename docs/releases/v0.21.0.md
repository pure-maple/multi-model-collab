# modelmux v0.21.0

## Cost Tracking, Documentation Overhaul & Repository Cleanup

### Highlights

- **Cost Tracking**: Automatic token usage extraction from Codex (`turn.completed`) and Gemini (`usageMetadata`) responses. New `costs.py` module with per-model pricing tables and aggregation. Query costs via `mux_history(costs=True)` or `modelmux history --stats --costs`.

- **Documentation Overhaul**: Complete rewrite of both English and Chinese READMEs to reflect current project state — all 6 MCP tools, A2A protocol, cost tracking, full CLI commands, updated architecture diagram, and feature summary table.

- **Repository Cleanup**: Removed 1,900+ lines of content not suitable for a public repository — raw AI consultation session logs (with session IDs and debug output), internal development notes, and code review process records. Replaced with a clean 47-line architecture consultation summary.

- **Security Hardening**: Added `.mcpregistry_*`, `.env`, `*.key`, `*.pem` to `.gitignore` to prevent accidental credential commits.

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.21.0
```
