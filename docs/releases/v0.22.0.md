## modelmux v0.22.0

Task decomposition, Provider/Model syntax, DashScope token usage, and A2A Push Notifications.

### Highlights

- **Task Decomposition**: `mux_dispatch(auto_decompose=True)` automatically breaks complex tasks into subtasks. Three-phase pipeline: planner analyzes task → wave executor runs subtasks in DAG topological order → merger combines results. Supports dependency declarations and parallel execution waves.
- **Provider/Model Syntax**: `mux_dispatch(provider="dashscope/kimi-k2.5")` and `mux_broadcast(providers=["dashscope/kimi-k2.5", "dashscope/MiniMax-M2.5"])` — same provider with different models can be broadcast in parallel. Unified `_parse_provider_spec()` parser.
- **DashScope Token Usage**: Extracts OpenAI-compatible `usage` field from DashScope API responses. New DashScope pricing in `costs.py` (Coding Plan subscription = $0). `estimate_cost()` supports `provider/model` format.
- **A2A Push Notifications**: Webhook callback on task completion. `PushConfig` dataclass, `_send_push_notification()` with httpx POST, integrated into both `tasks/send` and `tasks/sendSubscribe`. Agent Card declares `pushNotifications: true`.

### New Modules

- `decompose.py` — `DecompositionPlan`, `Subtask`, `parse_decomposition()`, `build_merge_prompt()`, DAG `execution_order()`

### Test Coverage

224 tests passing (35 new tests: 15 decomposition + 7 push notification + 6 broadcast/provider-spec + 7 DashScope costs)

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.22.0
```

## modelmux v0.22.0

任务自动拆解、Provider/Model 语法、DashScope 费用追踪、A2A 推送通知。

### 亮点

- **任务自动拆解**：`mux_dispatch(auto_decompose=True)` 自动将复杂任务拆分为子任务。三阶段流水线：planner 分析 → wave executor 按 DAG 拓扑排序并行/顺序执行 → merger 合并结果。支持依赖声明和并行执行波。
- **Provider/Model 语法**：`mux_dispatch(provider="dashscope/kimi-k2.5")` 和 `mux_broadcast(providers=["dashscope/kimi-k2.5", "dashscope/MiniMax-M2.5"])` —— 同一 provider 不同 model 可并行广播。统一的 `_parse_provider_spec()` 解析器。
- **DashScope 费用追踪**：从 DashScope API 响应中提取 OpenAI 兼容的 `usage` 字段。`costs.py` 新增 DashScope 全模型定价（Coding Plan 包月 = $0）。`estimate_cost()` 支持 `provider/model` 格式。
- **A2A 推送通知**：任务完成时 webhook 回调。`PushConfig` 数据类 + `_send_push_notification()` httpx POST，集成到 `tasks/send` 和 `tasks/sendSubscribe`。Agent Card 声明 `pushNotifications: true`。

### 新增模块

- `decompose.py` — `DecompositionPlan`、`Subtask`、`parse_decomposition()`、`build_merge_prompt()`、DAG `execution_order()`

### 测试覆盖

224 个测试通过（新增 35 个：15 拆解 + 7 推送 + 6 广播/规格解析 + 7 DashScope 成本）
