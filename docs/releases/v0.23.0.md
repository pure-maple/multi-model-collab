## modelmux v0.23.0

Web dashboard, webhook notifications, and benchmark suite.

### Highlights

- **Web Dashboard**: `modelmux dashboard` starts a local monitoring panel with dark-themed UI, 5-second auto-refresh. REST API endpoints: `/api/status`, `/api/history`, `/api/stats`, `/api/providers`, `/api/costs`. Shows active dispatches, provider health, cost summary, and history table.
- **Webhook Notifications**: Automatic POST to Slack/Discord/generic webhook URLs on dispatch completion. Auto-detects format from URL (Slack blocks, Discord embeds, generic JSON). Configure via `profiles.toml [notifications]` or `MODELMUX_WEBHOOK_URL` env var. Event filtering and non-blocking background threads.
- **Benchmark Suite**: `modelmux benchmark` runs standardized tasks across providers. 5 built-in tasks (code_review, code_generation, reasoning, summarization, translation). Keyword-based quality scoring, per-provider summary, JSON export.

### New Modules

- `dashboard.py` — Starlette web app with REST API and embedded HTML frontend
- `notifications.py` — Webhook notification system (Slack/Discord/generic)
- `benchmark.py` — Provider comparison benchmark framework

### New CLI Commands

- `modelmux dashboard [--port 41521]` — Start web monitoring dashboard
- `modelmux benchmark [--providers ...] [--tasks ...] [--output file.json]`

### Test Coverage

268 tests passing (+44 new: 14 dashboard + 16 notifications + 14 benchmark)

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.23.0
```

## modelmux v0.23.0

Web 监控面板、Webhook 通知、基准测试套件。

### 亮点

- **Web Dashboard**: `modelmux dashboard` 启动本地暗色主题监控面板，5 秒自动刷新。REST API 端点：`/api/status`、`/api/history`、`/api/stats`、`/api/providers`、`/api/costs`。展示活跃分发、provider 健康、费用汇总、历史记录。
- **Webhook 通知**: 分发完成时自动 POST 到 Slack/Discord/通用 webhook URL。从 URL 自动检测格式。通过 `profiles.toml [notifications]` 或 `MODELMUX_WEBHOOK_URL` 环境变量配置。支持事件过滤和后台非阻塞发送。
- **基准测试套件**: `modelmux benchmark` 跨 provider 标准化评测。5 个内置任务（代码审查、代码生成、推理、摘要、翻译）。关键词质量评分，per-provider 汇总，JSON 导出。

### 新增模块

- `dashboard.py` — Starlette Web 应用（REST API + 内嵌 HTML 前端）
- `notifications.py` — Webhook 通知系统（Slack/Discord/通用）
- `benchmark.py` — Provider 对比基准测试框架

### 新增 CLI 命令

- `modelmux dashboard [--port 41521]` — 启动 Web 监控面板
- `modelmux benchmark [--providers ...] [--tasks ...] [--output file.json]`
