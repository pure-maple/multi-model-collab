# v0.24.0 Release Notes

## Dashboard Trend Charts

The web dashboard now includes time-series visualizations:

- **Dispatch Volume** — stacked bar chart showing success/error counts per hour
- **Success Rate & Latency** — dual-axis line chart with success percentage and average duration

New API endpoint: `GET /api/trends?hours=24&bucket=60`

Returns bucketed data with: count, success, error, success_rate, avg_duration, cost, cumulative_cost, by_provider.

## Export / Reports

New CLI command: `modelmux export`

```bash
modelmux export --format csv --hours 24          # CSV for spreadsheets
modelmux export --format json --provider codex   # Structured JSON with stats
modelmux export --format md --output report.md   # Human-readable Markdown report
```

Supported formats:
- **CSV** — timestamp, provider, status, duration, task, tokens, model, run_id
- **JSON** — full entries + statistics + cost breakdown
- **Markdown** — Summary, Provider Breakdown table, Cost Summary, Recent Dispatches table
